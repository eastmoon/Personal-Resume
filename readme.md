# 個人履歷專案

+ Gitbook 電子書生成
> 使用 gitbook 生成個人履歷電子書

```
dockerw gitbook
```

+ eBook 電子書生成
> 使用 Markdown 為資料來排版並轉換成個人履歷電子書

```
dockerw ebook
```

+ website 履歷網站
> 使用 next.js 框架，建立讀取 Markdown 為資料來源的個人履歷網站

```
dockerw website
```
